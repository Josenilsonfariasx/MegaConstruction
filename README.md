<h1 align="center">Mega Construction</h1>

## :memo: Descrição
MegaConstruction:
Este projeto é uma pequena pagina para listagem de dados dos produtos e que pode se adequar a qualquer tipo de venda não so ha construções
## ⛹️‍♀️ Finalizado este Projeto Front-End segue o link ⬇️
* https://mega-construction.vercel.app/

## :books: Funcionalidades
* <b>Projetos e Stacks</b>: Usuario pode vizualizar os produtos e filtrar de acordo com categorias, preços ou nomes

## :rocket: Rodando o projeto
Para rodar o repositório é necessário clonar o mesmo, dar o seguinte comando para instalar as dependencias:
```
yarn
```
Ou:

```
npm install
```

Depois de instalar todas as dependencias execute o comando:
```
yarn dev
```
Ou:

```
npm dev
```
